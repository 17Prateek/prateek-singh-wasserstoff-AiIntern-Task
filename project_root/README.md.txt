# AI Image Segmentation and Object Analysis Pipeline

## Overview
This project implements a complete AI pipeline for image segmentation, object identification, text extraction, and object attribute summarization. It also includes a Streamlit app for testing and visualization.

## Folder Structure
- `data/`: Stores input images, segmented objects, and outputs.
- `models/`: Contains the models for segmentation, identification, text extraction, and summarization.
- `utils/`: Helper functions for preprocessing, postprocessing, data mapping, and visualization.
- `streamlit_app/`: Contains the Streamlit app.
- `tests/`: Unit tests for various components of the pipeline.
